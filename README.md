# AfriConnect 2026

# Présentation

AfriConnect 2026 est un site web dédié à un événement technologique africain. Il permet aux visiteurs de découvrir le programme, les intervenants, les partenaires, les tarifs, les informations pratiques et de s'inscrire en ligne.

Ce projet a été réalisé dans le cadre du module de Développement Web.

# Objectifs

* Présenter un événement professionnel.
* Permettre aux utilisateurs de consulter les informations importantes.
* Proposer un formulaire d'inscription.
* Offrir une interface moderne, responsive et facile à utiliser.

# Fonctionnalités

* Page d'accueil
* Présentation de l'événement
* Programme des conférences
* Présentation des intervenants
* Galerie d'images
* Liste des partenaires
* Tarifs d'inscription
* Formulaire d'inscription avec validation JavaScript
* FAQ
* Newsletter
* Compte à rebours
* Compteurs animés
* Bouton de retour en haut
* Menu responsive
* Mode sombre
* Animations au défilement

# Technologies utilisées

* HTML5
* CSS3
* JavaScript (ES6)

# Installation

1. Télécharger ou cloner le projet.
2. Ouvrir le dossier du projet.
3. Lancer le fichier **index.html** dans un navigateur web.

Aucune installation supplémentaire n'est nécessaire.

# Responsive Design

Le site est entièrement responsive et s'adapte aux ordinateurs, tablettes et smartphones grâce aux Media Queries en CSS.
